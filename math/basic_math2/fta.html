唯一分解定理

简介

对于一个大于 111 的整数 nnn，它要么本身是个质数，要么可以被唯一地分解成若干质因子的乘积的形式（质因子是以指数形式出现，且按照质因子从小到大的顺序排列）。

我们可以用数学公式将定理表示出来，即：

n=∏i=1kpiai=p1a1×p2a2×⋯×pkak n= \prod_{i=1}^{k} p_{i}^{a_{i}}=p_{1}^{a_{1}} \times p_{2}^{a_{2}}\times \cdots \times p_{k}^{a_{k}} n=i=1∏k​piai​​=p1a1​​×p2a2​​×⋯×pkak​​

其中 p1,p2,…,pkp_1, p_2, \ldots, p_kp1​,p2​,…,pk​ 为互不相同的质数，a1,a2,…,aka_1, a_2, \ldots, a_ka1​,a2​,…,ak​ 为正整数。

例如 181818 就可以表示为 21322^13^22132，它的质因子有 222、333，质因子 222 的个数为 111（指数为 111），质因子 333 的个数为 222（质数为 222）：

p1=2,a1=1p2=3,a2=2 p_1=2,a_1=1\\ p_2=3,a_2=2 p1​=2,a1​=1p2​=3,a2​=2

对于一个整数 nnn，它只有一种质因数分解的方式，不能有其他的分解方式。

例如 181818 它只有 21322^1 3^22132 这一种质因数分解的方式，而不能有其他的分解方式，如：

18≠223118≠2133⋯ 18 \neq 2^2 3^1 \\ 18 \neq 2^1 3^3 \\ \cdots 18=223118=2133⋯

唯一分解定理的应用非常广泛，下面我们来看看常见的三种应用。

一、质因数分解

  • 唯一分解定理 + 试除法。

质因数分解是将一个正整数分解成若干个质因子的乘积，它和因数分解类似，通过枚举 [2,n][2,\sqrt{n}][2,n​] 之间的数，找出我们想要的答案。

在因数分解中，当我们找到 nnn 的一个因子后，我们只是简单的将其存储，然后继续找下一个因子。而在质因数分解中，当我们找到 nnn 的一个因子后，我们需要令 nnn 不断除以该因子，直到 nnn 不在包含该因子。

下面以 121212 为例进行因数分解和质因数分解的演示。

因数分解

枚举 iii，i∈[2,12]i \in [2,\sqrt{12}]i∈[2,12​]：

i=2:12 mod 2=0,存储 2,122=6≠2,存储 6。i=3:12 mod 3=0,存储 3,123=4≠3,存储 4。 \begin{align*} i &= 2: & 12 \bmod 2 &= 0, \text{存储} \ 2,& \frac{12}{2} = 6 \neq 2, \text{存储} \ 6。 \\ i &= 3: & 12 \bmod 3 &= 0, \text{存储} \ 3, & \frac{12}{3} = 4 \neq 3, \text{存储} \ 4 。 \end{align*} ii​=2:=3:​12mod212mod3​=0,存储 2,=0,存储 3,​212​=6=2,存储 6。312​=4=3,存储 4。​

得到 nnn 的因子有 2,6,3,42,6,3,42,6,3,4。

质因数分解

枚举 iii，i∈[2,12]i \in[2,\sqrt{12}]i∈[2,12​]：

i=2i=2i=2：12 mod 2=012 \bmod 2=012mod2=0，存储 222，并令 121212 不断除以 222，直至除尽：

  • 12 mod 2=0⇒12÷2=612 \bmod 2 = 0 \Rightarrow 12 \div 2=612mod2=0⇒12÷2=6；
  • 6 mod 2=0⇒6÷2=36 \bmod 2=0 \Rightarrow 6 \div 2=36mod2=0⇒6÷2=3；
  • 3 mod 2=1≠03 \bmod 2=1 \neq 03mod2=1=0，停止。共除了 222 次，记录质因子 222 的指数为 222。

i=3i=3i=3：3 mod 3=03 \bmod 3=03mod3=0，存储 333，并令 333 不断除以 333，直至除尽：

  • 3 mod 3=0⇒3÷3=13 \bmod 3=0 \Rightarrow 3 \div 3=13mod3=0⇒3÷3=1；
  • 1 mod 3=1≠01 \bmod 3=1 \neq 01mod3=1=0，停止。共除了 111 次，记录质因子 333 的指数为 111。

得到 nnn 的质因子有 2,32,32,3。

在质因数分解的过程中，能被 nnn 整除的 iii，一定都是质数。

证明

假如 iii 是一个合数，那么它一定可以分解成多个质因子相乘的形式，这多个质因子同时也是 nnn 的质因子且比 iii 要小。但根据我们的步骤，比 iii 小的数在此之前的循环过程中已经被除尽了，所以 iii 不可能是合数，只可能是质数。

此外，在枚举结束后，我们还需要对剩余的 nnn （除尽质因子后的 nnn）作判断。如果剩余的 nnn 大于 111，则剩余的 nnn 也是原来的 nnn 的一个质因子，且指数为 111。

说明

原来的 nnn 中最多可能包含一个大于 n\sqrt{n}n​ 的质因子，且该质因子的指数最多为 111：

  • 如果不止包含一个，则它们相乘的结果为大于 nnn；
  • 如果指数大于 111，同理。

在我们枚举 2∼n2\sim \sqrt{n}2∼n​ 的过程中，该质因子不会被除尽，就会被遗留下来。所以如果剩余的数大于 111，那么它就是那个大于 n\sqrt{n}n​ 的质因子了。

C++
const int N = 1e5 + 10;
int p[N]; // p 数组用来存储 n 的质因子
int a[N];	// a 数组用来存储 n 的质因子的指数。
void work(int n){
    int cnt = 0; // cnt 记录质因子的个数, p[cnt] 表示 n 的第 cnt 个质因子
    for(int i = 2 ; i * i <= n ; i ++)
        if(n % i == 0){ 
          	// 这里的 i 一定是 n 的质因子
            p[++ cnt] = i; // 将 i 存入 p 数组中
            while(n % i == 0) {
              	n /= i;  // 利用循环将 n 的质因子 i 除尽
              	a[cnt] ++ ; // 记录除了多少次,除的次数 = 质因子的指数
            }
        }
    if(n > 1) p[++ cnt] = n , a[cnt] = 1; // 枚举结束后，如果 n > 1，则说明剩下的 n 也为原来的 n 的一个质因子，指数为 1
		for(int i = 1 ; i <= cnt ; i ++) cout << p[i] << " " << a[i] << '\n';
}

运行举例说明：

  • work(7) 返回的结果为：
7 1
  • work(12) 返回的结果为：
2 2
3 1
  • work(30) 返回的结果为：
2 1
3 1
5 1

二、求 nnn 的约数的个数（约数个数定理）

简介

约数（因数）个数定理是一个关于正整数的数学定理，它声称：对于任意正整数 nnn，它的约数个数等于它分解质因数后每个质因子的指数加 111 之积。

举个例子，如果我们要计算 121212 的约数个数，我们可以将 121212 分解质因数：

12=2231 12 = 2^2 3^1 12=2231

然后，对于每个质因子，我们将它的指数加 111 并相乘，得到：

(2+1)×(1+1)=6 (2+1)\times(1+1) = 6 (2+1)×(1+1)=6

因此，121212 的约数个数为 363636。

具体地，如果 nnn 的质因数分解式为：

n=p1a1×p2a2×⋯×pkak n=p_{1}^{a_{1}} \times p_{2}^{a_{2}}\times \cdots \times p_{k}^{a_{k}} n=p1a1​​×p2a2​​×⋯×pkak​​

那么 nnn 的约数个数就为：

∏i=1k(ai+1)=(a1+1)(a2+1)⋯(ak+1) \prod_{i=1}^{k} (a_{i}+1)=\left(a_{1}+1\right)\left(a_{2}+1\right) \cdots\left(a_{k}+1\right) i=1∏k​(ai​+1)=(a1​+1)(a2​+1)⋯(ak​+1)

至于原因，我们可以简单地用乘法原理来解释。

乘法原理

做一件事，完成它需要分成 nnn 个步骤，做第一步有 m1m_1m1​ 种不同的方法，做第二步有 m2m_2m2​ 种不同的方法，⋯\cdots⋯，做第 nnn 步有 mnm_nmn​ 种不同的方法，那么完成这件事共有 m1×m2×⋯×mnm_1\times m_2 \times \cdots \times m_nm1​×m2​×⋯×mn​ 种不同的方法。

nnn 可以分解质因数为 p1a1×p2a2×⋯×pkakp_{1}^{a_{1}} \times p_{2}^{a_{2}}\times \cdots \times p_{k}^{a_{k}}p1a1​​×p2a2​​×⋯×pkak​​，其中：

  • p1a1p_1^{a_1}p1a1​​ 的约数有 p10,p11,⋯ ,p1a1p_1^{0},p_1^{1},\cdots,p_1^{a_1}p10​,p11​,⋯,p1a1​​，共 (a1+1)(a_1+1)(a1​+1) 个；
  • p2a2p_2^{a_2}p2a2​​ 的约数有 p20,p21,⋯ ,p2a2p_2^0,p_2^1,\cdots, p_2^{a_2}p20​,p21​,⋯,p2a2​​，共 (a2+1)(a_2 + 1)(a2​+1) 个；
  • ⋯\cdots⋯
  • pkakp_k^{a_k}pkak​​ 的约数有 pk0,pk1,⋯ ,pkakp_k^{0},p_k^1,\cdots ,p_k^{a_k}pk0​,pk1​,⋯,pkak​​，共 (ak+1)(a_k+1)(ak​+1) 个。

根据乘法原理可得 nnn 的约数个数为 ∏i=1k(ai+1)=(a1+1)(a2+1)⋯(ak+1)\prod_{i=1}^{k} (a_{i}+1)=\left(a_{1}+1\right)\left(a_{2}+1\right) \cdots\left(a_{k}+1\right)∏i=1k​(ai​+1)=(a1​+1)(a2​+1)⋯(ak​+1)。

C++
void work(int n){
    int ans = 1; // 记录 n 的约数个数。初始化为 1(乘法)
    for(int i = 2 ; i * i <= n ; i ++)
        if(n % i == 0){ 
          	int cnt = 0;
            while(n % i == 0) {
              	n /= i;  
              	cnt ++; // 记录质因数 i 的指数
            }
         		ans *= (cnt + 1);
        }
    if(n > 1) ans *= (1 + 1);
  	cout << ans << '\n';
}

3

4
  • work(12) 返回的结果为：
6

三、求 nnn 的所有约数的和（约数和定理）

简介

约数（因数）和定理是一个关于正整数的数学定理，它声称：对于任意正整数 nnn，它的约数之和等于它分解质因数后每个质因子的所有次幂之和的乘积。

举个例子，如果我们要计算 121212 的约数之和，我们可以将 121212 分解质因数：

12=2231 12 = 2^2 3^1 12=2231

然后，对于每个质因子，我们将它的约数相加并相乘，得到：

(20+21+22)×(30+31)=7×4=28 (2^0 + 2^1 + 2^2)\times (3^0 + 3^1)= 7\times 4 = 28 (20+21+22)×(30+31)=7×4=28

因此，121212 的约数之和为 282828。

具体地，如果 nnn 的质因数分解式为：

n=p1a1×p2a2×⋯×pkak n=p_{1}^{a_{1}} \times p_{2}^{a_{2}}\times \cdots \times p_{k}^{a_{k}} n=p1a1​​×p2a2​​×⋯×pkak​​

那么 nnn 的约数之和就为：

(p10+p11+⋯+p1a1)(p20+p21+...+p2a2)⋯(pk0+pk1+⋯+pkak) (p_1^0+p_1^1+\cdots +p_1^{a_1})(p_2^0+p_2^1+...+p_2^{a2})\cdots(p_k^0+p_k^1+\cdots + p_k^{a_k}) (p10​+p11​+⋯+p1a1​​)(p20​+p21​+...+p2a2​)⋯(pk0​+pk1​+⋯+pkak​​)

至于原因，我们同样可以用乘法原理证明。

仔细观察不难发现：

p10+p11+⋯+p1a1p_1^0 + p_1^1 + \cdots +p_1^{a_1}p10​+p11​+⋯+p1a1​​

p20+p21+...+p2a2p_2^0+p_2^1+...+p_2^{a2}p20​+p21​+...+p2a2​

⋯\cdots⋯

pk0+pk1+⋯+pkakp_k^0+p_k^1+\cdots + p_k^{a_k}pk0​+pk1​+⋯+pkak​​

均是等比数列。因此，我们可以套用等比数列求和公式，进一步得到：

∏i=1kpiai+1−1pi−1=p1a1+1−1p1−1×p2a2+1−1p2−1×⋯×pkak+1−1pk−1 \prod_{i=1}^{k} \dfrac{p_{i}^{a_{i}+1}-1}{p_{i}-1} = \dfrac{p_1^{a_1+1}-1}{p_1-1}\times \dfrac{p_2^{a_2+1}-1}{p_2-1}\times \cdots \times \dfrac{p_k^{a_k + 1} - 1}{p_k - 1} i=1∏k​pi​−1piai​+1​−1​=p1​−1p1a1​+1​−1​×p2​−1p2a2​+1​−1​×⋯×pk​−1pkak​+1​−1​
